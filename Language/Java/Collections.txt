### Collections Framework
- 데이터 군을 저장하는 클래스들을 표준화한 설계
- List, Set, Map

<br>

### List
- 순서가 있는 데이터의 집합
- 데이터의 중복 허용
- Collection 인터페이스를 상속
- 구현 클래스 : ArrayList, LinkedList, Stack, Vector

<br>

### Set
- 순서가 없는 데이터의 집합
- 데이터의 중복 불가
- 구현 클래스 : HahsSet, TreeSet


<br>

### Map
- Key와 Value의 쌍으로 이루어진 데이터 집합
- 순서가 유지되지 않는 데이터 집합
- Key 데이터는 중복 허용
- Value 데이터는 중복 불가
- 구현 클래스 : HashMap, TreeMap, HashTable, Properties
